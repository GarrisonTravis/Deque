# Deque

This repository contains the code for a templated Deque created with a doubly linked list data structure.

Operations:
-----------
  - Two constructor choices
  - getSize O(1)
  - empty   O(1)
  - push_front O(1)
  - push_back  O(1)
  - pop_front  O(1)
  - pop_back   O(1)
  - front      O(1)
  - back       O(1)
  - clear      O(n)
  - print      O(n)
